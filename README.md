# BCT
BCT is Bash Scripting to backup validator config folder in IBC testnet nodes
```bash
.
├── addrbook.json
├── app.toml
├── client.toml
├── config.toml
├── genesis.json
├── node_key.json
└── priv_validator_key.json
```

# Run
```
curl -O https://raw.githubusercontent.com/Tnodes/BCT/main/run.sh && chmod +x run.sh && ./run.sh
```
