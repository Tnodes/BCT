# BCT
BCT is Bash Scripting to backup validator config folder in IBC testnet & mainnet nodes
```bash
.
├── addrbook.json
├── app.toml
├── client.toml
├── config.toml
├── genesis.json
├── node_key.json
└── priv_validator_key.json
```

# Feature
* Directory listing to download backup config in another devices
